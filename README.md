# 🍔 Gourmet Food Landing Page (The Cheese Decker)
Welcome to The Cheese Decker! A high-performance, visually stunning landing page for a premium burger brand. This project focuses on modern UI/UX principles to drive user engagement and conversions.

---
###  Gourmet Food Landing Page (Internship Task)
*Speciality:*   A fully responsive, modern food-delivery style landing page featuring "The Cheese Decker" branding.
* *Key Features:*   Animated Floating Hero image, Responsive Menu Grid, Newsletter Subscription, and Interactive Contact Form.
* *Use Case:*   Perfect for showcasing front-end design skills, specifically for the E-commerce and Food & Beverage industry.
* *Technologies:*  HTML5, CSS3 (Flexbox & Grid), JavaScript (ES6+), FontAwesome, and Google Fonts (Jacques Francois Shadow).
* *Demo:* [View Live Demo](https://cheese-decker.netlify.app/)

## 🛠️ Technical Implementation
* *Dynamic Navigation:* Implemented a custom mobile-responsive hamburger menu using pure CSS and JS toggle logic.

* *Floating Animations:*  Used CSS @keyframes to create a floating effect for the main burger image, making the page feel alive.

* *Smart UI Elements:*  Integrated a hover-responsive menu card system that provides a smooth user experience.

* *Interactive Forms:*  Functional JavaScript validation for the Newsletter (20% Off logic) and Contact Us sections.
